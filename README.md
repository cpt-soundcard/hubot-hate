hubot-hate
==========

This script helps quell the hate by asking the age old question: "Maybe you should hustle as hard as you hate?"

## Installation

Add **hubot-hate** to your `package.json` file:

```json
"dependencies": {
  "hubot-hate": ">= 0.0.1",
}
```

Add **hubot-hate** to your `external-scripts.json`:

```json
["hubot-hate"]
```

Run `npm install`
